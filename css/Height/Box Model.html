Mendemonstrasikan Model Kotak

Model kotak CSS pada dasarnya adalah sebuah kotak yang membungkus setiap elemen HTML. Terdiri dari: batas, padding, margin, dan konten sebenarnya.

Teks ini adalah isi kotak. Kami telah menambahkan padding 50px, margin 20px dan batas hijau 15px.
